jQuery.lightValidate by manji6

"jQuery.lightValidate" is very simple, a JavaScript library for performing simple processing and validating.

jQuery.lightValidate は簡単にバリデーションを設定できるjQueryプラグインです。 特徴として、このライブラリ自身にエラー表示処理が無いところです。。。 どういうことかというと、エラーを検知する仕組みはあるんですが、そのエラーに対してどうするか？は超自由にカスタマイズできるようになっています。

  1. まずはjQuery本体をロード後に当ライブラリを読み込んでください
  2. バリデーション内容をobject型で定義します。 
    var lightValidateCond = {[
	{
		// key "element" にはチェックしたい項目のCSSセレクタを設定(必須)
		element: "input[name=username]",
		// key "type" にはチェックタイプを定義。種類に関しては後述(必須)
		type   : "require"
	},{
		element: "input[name=type]",
		// key "type" に対して複数定義する場合はarray型にして定義
		// (この場合は必須要素で、key "group" に定義した要素の中を満たす事
		type   : ["require","group"],
		group  : ["user","administrator"]
	},{
		element: "input[name=date]",
		type   : ["require","regex"],
		regex  : /^[1-9]{1}[0-9]{3}[\/-\.]+[0-1]+[0-9]{1}[\/-\.]+[0-3]+[0-9]{1}$/
	}
]};
  3. 定義したobjectを$.lightValidate()の第1引数に渡す。第2引数はエラー処理を記述出来るcallback関数となる。また、当関数の戻り値はエラー有無がbool型で渡されるので、その戻り値を使ったバリデーション後の後処理を記述可能。 
    // 例えばエラー時にこれ以降の処理をストップする場合
if($.lightValidate(lightValidateCond,function(errorArray){
	// エラー時はerrorArrayにエラー内容が返却される。
	// それぞれエラー内容はobject型で"element","message"が含まれる。

}) === false){
	// エラーバリデーション処理後の処理を此処に書く
	return false;
}
